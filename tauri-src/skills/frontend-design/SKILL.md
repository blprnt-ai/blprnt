# Frontend Design Skill

## Purpose
Produce cohesive, accessible UI updates that align with existing design patterns.

## When to Use
- Designing new UI layouts or sections
- Refining visual hierarchy, spacing, or typography
- Reviewing UI consistency before implementation
- Planning component styling and interaction behavior

## Workflow
1. **Set directional intent** for the UI and commit to it.
2. **Define the signature system** (domain, color world, signature, defaults).
3. **Specify typography** with clear hierarchy and scale.
4. **Create CSS variables** for color, spacing, radius, and layering tokens.
5. **Compose layout and interactions** with minimal overrides.
6. **Validate layering, accessibility, and responsiveness** in context.
