# Component Redesign Example

## Context
Redesign a card component to improve scanability.

## Proposed Structure
- Header: title + status badge
- Body: summary text with constrained width
- Footer: primary action + secondary link

## Interaction States
- Hover: subtle border highlight
- Focus: visible outline
- Disabled: reduced opacity and no hover effect

## Notes
- Use existing spacing tokens for padding.
- Keep badge color aligned with status palette.
