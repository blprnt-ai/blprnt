# Layout Audit Example

## Context
A settings panel feels cramped and unclear.

## Findings
- Primary action blends with secondary buttons.
- Section spacing is inconsistent.
- Headings are visually similar to body text.

## Recommendations
- Increase primary button weight and contrast.
- Normalize section spacing to a consistent scale.
- Use a larger heading style with stronger weight.

## Example Notes
- Align actions to the same baseline.
- Add a divider between major groups.
