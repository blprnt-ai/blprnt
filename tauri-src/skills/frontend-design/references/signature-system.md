# Signature System

## Required Outputs
Produce all four elements and keep them consistent across the UI:

1. **Domain**: What the interface is about (work, focus, calm, urgency).
2. **Color World**: Primary palette + support neutrals + accent.
3. **Signature**: Distinctive treatment (radius, shadow, icon style, motion).
4. **Defaults**: Baseline typography, spacing scale, and border styles.

## Rules
- Every screen should express the same domain and signature.
- Defaults must be usable without extra overrides.
- Color world should define both action and surface states.
